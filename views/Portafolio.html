TECNOLOGIAS IMPLEMENTADAS

Aquí encontrarás las herramientas usadas para el desarrolo web de yamix

Backend con Node.js y Express

 API REST utilizando Node.js con Express,  Se implementó un modelo en capas, organizando el código en controladores, servicios y rutas para una mejor escalabilidad.

Base de Datos MySQL

Se utilizó MySQL para gestionar la información del club, incluyendo estudiantes, asistencia, clases y eventos. La conexión se estableció con mysql2, garantizando consultas optimizadas y seguras.

Integración y Seguridad

Se implementaron JWT  para autenticación de usuarios y reCAPTCHA para evitar bots en los formularios. Además, se utilizó Postman para probar y documentar la API REST.

Frontend EJS y FullCalendar

Se íntegro FullCalendar para la gestión visual de eventos y asistencia. La interacción con la API permite mostrar y administrar clases y eventos en tiempo real.

Proy​ecto yam​ix

La solución que le brindamos a Yamix es un aplicativo web-mobile que optimiza la gestión de inscripciones y el control de asistencia de los estudiantes.

Acerca del proyecto a desarrollar.

Plan de desarrollo.

La necesidad de la academia era optimizar su gestión operativa, mediante un sistema que controlara usuarios, asistencias, eventos, clases. Se le diseñó un aplicativo web completo que cubría todo lo planteado. Tiempo de desarrollo: 6 meses.
